# Desarrollo-y-Evaluaci-n-de-Modelos-de-Regresi-n-y-Clasificaci-n
En este proyecto se realiza un proceso de limpieza y transformación de datos con el objetivo de optimizar el rendimiento del modelo predictivo. Se identifican y eliminan variables con baja correlación y alta multicolinealidad para reducir la dimensionalidad del dataset. Posteriormente, se entrenan y comparan modelos de regresión utilizando métricas como R² para evaluar su desempeño.
